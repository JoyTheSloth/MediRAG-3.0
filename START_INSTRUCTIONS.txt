# How to start MediRAG manually in VS Code

To start the project, you need two separate terminal windows in VS Code.
One for the Backend API, and one for the Frontend UI.

---

### Terminal 1: Run the Backend (FastAPI)
1. Open a new terminal in VS Code (Ctrl + Shift + `)
2. Change into the Backend directory:
   cd "e:\MediRag 2.0\Backend"

3. Run the FastAPI server:
   python -m uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000

The backend should now say it is running on http://127.0.0.1:8000
    
---

### Terminal 2: Run the Frontend (React / Vite)
1. Open a SECOND terminal in VS Code (Keep the backend running!)
2. Change into the Frontend directory:
   cd "e:\MediRag 2.0\Frontend"

3. (Optional) Install packages if you haven't recently:
   npm install

4. Start the frontend development server:
   npm run dev

The frontend UI will now be available in your browser at:
http://localhost:5173

---
If you run into missing module errors in the backend, run `python -m pip install -r requirements.txt`.
